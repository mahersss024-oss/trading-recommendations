## 🚀 طريقة التشغيل

### تشغيل محلي:
```bash

pip install -r requirements.txt

# 2. تشغيل التطبيق

# أو استخدام السكريبت المجهز

```

### تشغيل على Streamlit Cloud:
1. رفع الملفات إلى GitHub

3. تحديد `app_streamlit.py` كملف رئيسي
